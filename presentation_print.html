Как составлен список

1

Поиск через справочники

yellowpages.uz и top.uz — официальные каталоги с 274+ стоматологиями, 200+ салонами, 76+ кафе. Фильтрация: только компании без поля «Сайт» в карточке.

2

Верификация через поисковики

Каждую компанию проверяли в Google — ищем домен, официальный сайт. Если в топе только каталоги (2GIS, Yandex.Maps, Yellow Pages) — сайта нет.

3

Ранжирование по вероятности

Оценка 25–90% по критериям: категория бизнеса, наличие контактов, цифровая активность (Instagram, Facebook), ценовой сегмент услуг.

4

Формирование предложения

Для каждой категории — отдельный шаблон письма + живое демо. Под стоматологии готов пример с онлайн-записью, врачами, прайсом и отзывами.
